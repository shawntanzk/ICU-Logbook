ICU Logbook

Registration Validator

Authorised verifiers can check whether a medical registration number is associated with an active ICU Logbook account. No personal data is returned.

Issued separately to authorised validators. Do not share.

Spaces and capitalisation are normalised automatically.

🔒 Privacy & Security: This tool sends the registration number to our server over HTTPS. The server computes an HMAC-SHA256 hash and compares it to stored hashes — the original number is never stored in our database, and this tool cannot reveal the registrant's name, email, or any clinical data.
